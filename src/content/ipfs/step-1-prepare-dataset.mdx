Before you start, gather everything you want to share into a single folder, for example:

```
my-dataset/
├── README.md
├── blocks/
├── transactions/
└── logs/
```

The data can feed directly into one of our decentralisation-analysis tools, we accept it in that raw format too — see our <Link to={dataFormatsRoute.to} className="link link-primary font-medium">Data Format Templates</Link> for the schema, by layer.

We also recommend including a `metadata.json` file describing the dataset. You can start from our template:

<a href={basePath + "/downloads/ipfs/metadata.template.json"} download className="link link-primary font-medium">
  Download metadata.template.json
</a>

```json
{
  "project": "",
  "network": "",
  "node_version": "",
  "collection_date": "",
  "dataset_version": "",
  "description": ""
}
```
